# HiFloat8 / HiF8 Complete 256-Value Table

本文按 Ascend HiFloat8 论文规则枚举 `0x00` 到 `0xFF` 的全部 256 个编码。

## 解码规则

HiF8 总体格式：

```text
sign | dot field | exponent field | mantissa field
```

Dot prefix code：

| Dot | Prefix | Exponent bits | Mantissa bits | Exponent range |
| --- | --- | ---: | ---: | --- |
| D4 | `11` | 4 | 1 | `±[8, 15]` |
| D3 | `10` | 3 | 2 | `±[4, 7]` |
| D2 | `01` | 2 | 3 | `±[2, 3]` |
| D1 | `001` | 1 | 3 | `±1` |
| D0 | `0001` | 0 | 3 | `0` |
| DML | `0000` | 0 | 3 | denormal-like |

Normal 编码：

```text
value = (-1)^S * 2^E * 1.M
```

当 Dot 不是 DML 时，exponent field `Em` 使用 sign-magnitude exponent：

```text
Em = {Se, Mag[2:end]}
完整 magnitude = {1, Mag[2:end]}
Se = 0 -> E = +magnitude
Se = 1 -> E = -magnitude
```

DML 编码：

```text
value = (-1)^S * 2^(M - 23), M in [1, 7]
S=0, M=0 -> zero
S=1, M=0 -> NaN
```

Infinity：

```text
0x6F -> +Inf
0xEF -> -Inf
```
